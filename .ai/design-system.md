# Design System (Tavora)

## Color Palette

| Token | Hex | Usage |
|---|---|---|
| `--color-black` | `#0A0A0A` | Primary background, deep surfaces |
| `--color-white` | `#FAFAFA` | Pure white text/icons |
| `--color-gold` | `#C9A96E` | Champagne Gold - primary accent, CTAs |
| `--color-gold-light`| `#D4BC8B` | Hover states |
| `--color-gold-dark` | `#B08D4C` | Pressed states |
| `--color-silver` | `#C0C0C0` | Secondary accents, borders |
| `--color-gunmetal` | `#2C3539` | Secondary backgrounds, cards |
| `--color-charcoal` | `#1A1A1A` | Slightly raised surfaces |
| `--color-warm-gray` | `#8A8A8A` | Muted text, disabled states |

## Typography
- **Headings (Display/Hero/H1-H4):** `Playfair Display` (Serif, conveys heritage and luxury)
- **Body & UI (Text/Buttons/Labels):** `Inter` (Sans-serif, clean readability)

## Spacing & Layout
- Generous whitespace.
- Floating cards with soft, subtle shadows (never harsh).
- Rounded corners: `0.5rem` (8px) for cards, `0.25rem` (4px) for smaller elements.

## Animations
- Framer Motion for micro-interactions.
- Fade-ins on scroll.
- Subtle scale-up on image hover (1.02 to 1.05 max).
- Easing should feel cinematic (smooth, not bouncy).
